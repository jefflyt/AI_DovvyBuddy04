# src/lib

Shared utilities and core logic for DovvyBuddy.

## Contents (to be added in future PRs)

- **model-provider/** — LLM provider interface with Groq and Gemini implementations
- **rag/** — Retrieval-Augmented Generation pipeline (embedding, chunking, search)
- **session/** — Session management utilities
- **validation/** — Input validation and sanitization
- **utils/** — General-purpose helper functions

## Usage

Import from `@/lib` using the TypeScript path alias:

```typescript
import { someUtil } from '@/lib/utils'
```
